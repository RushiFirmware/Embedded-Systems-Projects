# DC Motor Control

## Description
This project demonstrates the control of a DC motor using Arduino and a motor driver module.

## Components Used
- Arduino Uno
- DC Motor
- L293D Motor Driver Module
- Power Supply
- Connecting Wires

## Features
- Start and Stop Motor
- Control Motor Direction
- PWM Speed Control

## Language
- Arduino C++
